# Mini SaaS CI/CD Project

## Overview

Production-style multi-service deployment project using:

- Node.js (Express)
- PostgreSQL
- Redis
- Docker Compose
- GitHub Actions CI/CD

## Features

- Multi-container deployment
- Health check endpoint
- Auto CI validation on push
- Environment variable management
- Production-ready structure

## Services

- Backend API
- PostgreSQL Database
- Redis Cache

## CI/CD Flow

Git Push → GitHub Actions → Install → Run → Health Check

## Endpoints

- /health
- /users

## Tech Stack

Docker, GitHub Actions, Node.js, PostgreSQL, Redis

## Use Case

Built to simulate real SaaS deployment projects for DevOps freelancing and production deployment.
